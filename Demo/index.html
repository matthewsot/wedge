Wedge

A simple, open-source lightbox.

Wedge is a drop-dead-simple lightbox written in Typescript.
You can use it to show anything from images and Youtube videos to custom divs.

Like it? Using it requires just two lines of HTML:

<!-- JQuery -->
<script type="text/javascript" src="//ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"></script>
<!-- Get this from Github -->
<script type="text/javascript" src="/Javascript/Wedge.js"></script>

Wedge.js comes with two default animations (FadeAnimation and SlideAnimation).
If you want to use the extra animations in Animations.js (like ScaleAnimation and RotateAnimation), you'll need three more lines:

<!-- GSAP animations library -->
<script type="text/javascript" src="//cdnjs.cloudflare.com/ajax/libs/gsap/1.11.6/TweenMax.min.js"></script>
<!-- jquery.gsap plugin - Get this from Github -->
<script type="text/javascript" src="jquery.gsap.min.js"></script>
<!-- You can get this from Github as well -->
<script type="text/javascript" src="/Javascript/Animations.js"></script>
